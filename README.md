# Mobile Appointment Booking App UI

## Project Overview

This project was completed as part of the Future Interns UI/UX Design Internship Program.

The objective was to design a mobile-first appointment booking application that helps local service businesses manage appointments efficiently while providing a smooth and user-friendly booking experience for customers.

---

## Business Chosen

### SparkClean Home Cleaning Services

Location: Kochi, Kerala

SparkClean is a local home cleaning service that currently relies on phone calls and messaging platforms for appointment scheduling.

### Existing Challenges

* Missed bookings
* Scheduling conflicts
* Slow appointment confirmation
* Poor customer convenience

This mobile app was designed to solve these problems through a simple digital booking flow.

---

## Target Users

* Homeowners
* Apartment residents
* Working professionals
* Customers seeking convenient cleaning services

These users want a quick and hassle-free way to schedule appointments.

---

## Booking Flow

### 1. Onboarding Screen

Purpose:

* Introduce the service
* Explain the app's value
* Encourage users to begin booking

### 2. Service Selection Screen

Users can select:

* Deep Cleaning (₹999)
* Kitchen Cleaning (₹799)
* Bathroom Cleaning (₹699)
* Move-Out Cleaning (₹1299)

### 3. Date & Time Selection Screen

Users choose:

* Preferred appointment date
* Available time slot

This reduces booking confusion and improves scheduling accuracy.

### 4. Booking Confirmation Screen

Displays:

* Selected service
* Appointment date
* Time slot
* Total amount

Provides immediate booking confirmation.

---

## UX Design Decisions

### Mobile-First Approach

The interface was designed specifically for smartphone users with touch-friendly interactions.

### Simple Navigation

The booking flow follows a clear step-by-step process to reduce friction and cognitive load.

### Visual Hierarchy

Important actions such as booking and confirmation are highlighted using clear call-to-action buttons.

### Clear Pricing

Service prices are displayed upfront to improve transparency and user trust.

### Minimal Booking Steps

The user can complete the booking process in four simple screens.
